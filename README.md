# 🏃‍♂️ 운동 로그 트래커 (Workout Tracker)

MUI(Material-UI)와 React를 활용한 종합적인 운동 로그 시각화 웹 애플리케이션입니다.

## ✨ 주요 기능

- **📅 월간/연도별 달력 뷰**: 운동 기록을 달력 형태로 한눈에 확인
- **🎯 이모지 기반 시각화**: 각 운동 종류별로 이모지와 색상으로 구분
- **📊 상세 통계 및 차트**: 운동별 진행 상황을 다양한 차트로 시각화
- **🌙 다크/라이트 모드**: 사용자 선호에 따른 테마 전환
- **📱 반응형 디자인**: 모바일과 데스크톱에서 모두 최적화된 UI
- **📈 월별/연도별 통계**: 장기적인 운동 진행 상황 분석
- **🔥 연속 운동 스트릭**: 현재/최장/월간 스트릭 및 다음 마일스톤 추적
- **📊 월간 진행률 바**: 운동별 목표 달성률 및 자동 목표 설정
- **📈 운동 추이 그래프**: 일별 운동 횟수, 타입별 분포, 일별 추이 차트
- **👆 직관적인 인터페이스**: 날짜 클릭으로 운동 상세 정보 확인

- 🏊‍♂️ **수영** (Swimming) - 파란색
- 🏃‍♂️ **런닝** (Running) - 초록색  
- 🚴‍♂️ **자전거** (Cycling) - 보라색
- 🏢 **계단오르기** (Stair Climbing) - 주황색

- 🏋️‍♂️ **헬스** (Gym) - 회색
- 🏄‍♂️ **서핑** (Surfing) - 청록색
- ⛳ **골프** (Golf) - 연두색

## 🚀 시작하기

### 필수 요구사항

- Node.js (v14 이상)
- npm 또는 yarn

### 설치 및 실행

1. **저장소 클론:**
```bash
git clone [repository-url]
cd workout
```

2. **의존성 설치:**
```bash
npm install
```

3. **개발 서버 실행:**
```bash
npm start
```

4. **브라우저에서 [http://localhost:3000](http://localhost:3000) 접속**

### 빌드 및 배포

```bash
# 프로덕션 빌드
npm run build

# 테스트 실행
npm test
```

## 🛠️ 기술 스택

- **React 18**: 사용자 인터페이스 구축
- **Material-UI (MUI) v5**: 디자인 시스템 및 컴포넌트
- **@mui/x-date-pickers**: 날짜 선택 컴포넌트
- **date-fns**: 날짜 처리 및 포맷팅 (한국어 로케일 지원)
- **Emotion**: CSS-in-JS 스타일링
- **Recharts**: 데이터 시각화 차트 라이브러리

## 📁 프로젝트 구조

```
src/
├── components/
│   ├── WorkoutCalendar.js      # 메인 달력 컴포넌트
│   ├── MonthlyProgressBar.js   # 월간 진행률, 스트릭, 그래프
│   └── WorkoutDetailView.js    # 운동 상세 통계 및 차트 뷰
├── data/
│   └── workoutData.js          # 운동 데이터 관리 및 API
├── App.js                      # 메인 앱 컴포넌트 (라우팅, 테마 관리)
├── index.js                    # 앱 진입점
└── index.css                   # 전역 스타일
```

## 🔧 주요 컴포넌트 설명

### 📅 WorkoutCalendar
- 월간/연도별 달력 형태로 운동 기록 표시
- 운동 종류별 색상 코딩
- 날짜 클릭 시 상세 정보 표시 (운동이 있는 날짜만)
- 월별/연도별 뷰 전환 지원
- 반응형 디자인으로 모바일 최적화

### 📊 MonthlyProgressBar
- **월간 진행률 요약**: 전체 운동일 달성률, 총 운동 횟수
- **운동별 진행률**: 각 운동 타입별 목표 대비 진행 상황
- **연속 운동 스트릭**: 현재/최장/월간 스트릭, 다음 마일스톤
- **자동 목표 설정**: 이전 달 기록 기준으로 자동 목표 제안
- **운동 추이 그래프**: 일별 총 운동 횟수, 타입별 분포, 일별 추이

### 📊 WorkoutDetailView
- 선택된 운동의 상세 통계 제공
- 월별/연도별 진행 상황 차트
- 다양한 차트 타입 (선형, 막대, 파이, 영역)

## 🔧 커스터마이징

### 새로운 운동 종류 추가

`src/data/workoutData.js` 파일의 `workoutTypes` 객체에 새로운 운동을 추가:

```javascript
const workoutTypes = {
  // 기존 운동들...
  newWorkout: { 
    emoji: '🎯', 
    label: '새운동', 
    color: '#your-color'
  }
};
```

### 데이터베이스 연동

현재는 로컬 스토리지를 사용하고 있습니다. 실제 데이터베이스와 연동하려면 `src/data/workoutData.js`의 함수들을 수정하세요.

## 📊 데이터 구조

```javascript
{
  date: "2024-01-15",
  type: "running",
  duration: 30,    // 분 단위
  distance: 5,     // km 단위 (런닝, 자전거 등)
  sets: 3,         // 세트 수 (맨몸운동, 헬스 등)
  reps: 15         // 횟수 (맨몸운동, 헬스 등)
}
```

## 🚀 사용 방법

### 📅 운동 기록 확인
1. **월별 보기**: 월간 달력에서 운동 기록 확인
2. **연도별 보기**: 연도별 전체 운동 통계 확인
3. **운동 상세**: 운동 종류 클릭 시 상세 통계 확인
4. **날짜 상세**: 운동이 있는 날짜 클릭 시 해당 날짜 운동 정보 확인

### 📊 통계 확인
- **월간 진행률**: 월별 운동 달성률 및 목표 진행상황
- **연속 기록**: 각 운동별 연속 운동 일수
- **운동 추이**: 일별/월별 운동 패턴 분석
- **종합 통계**: 전체 운동일, 총 운동 횟수, 월평균 등
